This is an app that runs as a server on the pc and you can connect to it over network via ip on any device. Simple tracking of daily lines for each batters and pitchers and scoreboard.
VMix Live JSON Endpoints:
  Scoreboard:   http://[YOUR_PC_IP]:5000/api/scoreboard.json
  Home Batter:  http://[YOUR_PC_IP]:5000/api/homebatter.json
  Away Batter:  http://[YOUR_PC_IP]:5000/api/awaybatter.json
